# Bob Ross Episode Index

Master searchable list of all 403 "Joy of Painting" episodes with semantic tags and direct links.

---

## How to Use This Index

- **Browse by Season**: Scroll to find episodes by season and episode number
- **Search by Tags**: Use your browser's find function (Ctrl+F / Cmd+F) to search for:
  - Composition archetypes (e.g., "mountain reflection", "forest corridor")
  - Palette identities (e.g., "dark woodland", "warm sunset")
  - Depth styles (e.g., "tunnel depth", "atmospheric fade")
  - Lighting types (e.g., "backlit glow", "diffuse")
  - Featured elements (e.g., "waterfall", "cabin", "ocean")
- **Access Full Episodes**: Click episode links to view complete semantic analysis
